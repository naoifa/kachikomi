## Phase 8｜コピーライティング（決定稿の要旨）

### Hero（QUEST）
- Qualify+Understand: バッジ「中小企業・スタートアップ・EC事業者のためのWeb制作」
- h1: **「そのホームページ、成果を生んでいますか。」**
- サブ: デザインの美しさは、手段です。AI Web Studioは問い合わせ・売上・採用——事業の成果から逆算して、ホームページ・LP・ECサイトを設計するWeb制作会社です。
- CTA: [無料相談する]（主）/ [無料サイト診断を受ける]（副）
- 信頼バー: 対応実績120社+ / 平均CVR改善2.4倍 / Lighthouse平均98 / 最短納期2週間

### Problem（PAS: Problem/Agitation）
h2「こんな『もったいないサイト』になっていませんか。」
1. 作って終わりで放置 — 更新されない情報が、会社の信頼を静かに削っている
2. 綺麗だけど売れない — 動線設計のないデザインは、離脱の入り口
3. スマホで遅い・見づらい — 表示に3秒かかるだけで、訪問者の半分は去る

### Solution（PASのSolution + 3本柱）
h2「答えはシンプル。『成果から逆算』して作ること。」
1. 戦略が先、デザインは後 — ペルソナとCV設計から始める
2. AI × プロフェッショナル — AIで速く、プロが仕上げる。品質と価格の両立
3. 公開してからが本番 — 保守運用・改善提案まで伴走

### Feature（サービス7つ）
ホームページ制作 / LP制作 / ECサイト制作 / SEO対策 / MEO対策 / 保守運用 / AI導入支援

### Benefit（4P: Picture/Promise/Prove/Push）

### Evidence（数値）
