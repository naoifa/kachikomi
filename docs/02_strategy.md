# 戦略ドキュメント（Phase 3–5）

## Phase 3｜競合分析

対象: 中小企業向けWeb制作会社の典型的サイト（大手制作会社・地域制作会社・フリーランス、クラウドソーシング）

| 競合タイプ | 強み | 弱み | 当社の突き方 |
|---|---|---|---|
| 大手制作会社 | 実績量・ブランド | 価格不透明（最低100万〜）・小回りが利かない・営業が分業で遅い | **料金3プラン明示**・最短2週間の速度 |
| 地域の中小制作会社 | 対面安心感 | サイト自体が古い・成果指標を語らない・スマホ表示が遅い | **自社サイト自体が実証**（表示速度・デザインが営業ツール） |
| フリーランス/クラウドソーシング | 安い | 品質のばらつき・保守が続かない・ディレクション不在 | **チーム体制と保守運用**を明示、AI活用で大手品質×個人価格帯 |

### 業界共通の弱点（=差別化ポイント）
1. **料金を隠す** → 3プラン+含まれる内容を明記
2. **「作って終わり」** → 保守運用・改善提案をサービスの柱に
3. **成果を語らない（デザイン自慢のみ）** → 全コピーをCV・売上・集客の言葉で書く
4. **AI活用が言葉だけ** → AI導入支援をメニュー化し、制作プロセスでのAI活用を明記
5. **自社サイトが重い** → Lighthouse 100 / Core Web Vitals実測値を掲載できる設計

## Phase 5｜SEO設計

### キーワードマップ
| 種別 | キーワード | 対応箇所 |
|---|---|---|
| 指名 | AI Web Studio | title/OGP/Organization JSON-LD |
| 主要 | ホームページ制作 / Web制作会社 | title・h1・Service JSON-LD |
| 主要 | LP制作 / ECサイト制作 | Featureセクションh3・Service JSON-LD |
| 収益 | ホームページ制作 料金 | Priceセクション+FAQ |
| ロングテール | ホームページ 作り直し 費用 / LP制作 CVR | FAQ（FAQPage JSON-LD） |
| 先進 | AI導入支援 / Next.js 制作 | Feature・FAQ |

### 技術SEO実装項目
- `metadata`（title 32字以内 / description 120字）+ canonical + OGP + Twitter Card
- `robots.txt` / `sitemap.xml`（App Routerの`robots.ts`/`sitemap.ts`で自動生成）
- JSON-LD: `Organization` `WebSite` `Service` `FAQPage`
- セマンティックHTML（h1は1つ、セクションはlandmark+aria-labelledby）
- 全画像（SVG含む）にalt、装飾はaria-hidden
- Core Web Vitals: 画像ゼロ方針でLCP=テキスト、CLS=0設計（レイアウトシフト要素なし）
